# Vue Monty Hall 
Monty Hall Problem made with Vue

## Screenshots
![Login](demo/demo.gif)

## Dependencies
- VueJS

## Dev dependencies
To run on dev environment run:

```sh
npm install @vue/cli
npm install
npm run serve
```
> The Monty Hall Problem will run on  http://localhost:8080
